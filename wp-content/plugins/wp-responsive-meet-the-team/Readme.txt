== Description ==

WP Responsive Meet The Team plugin allows you to easily create your team or staff pages. Simply create your team and add them to you page to display for all screen sizes with a simple shortcode.

In this free version you have access to the circular team layout style, however in the PRO version you will have access to all five styles and be able to change the styles at a click of a button without having to re-setup the team again.

= Free version features include: =
* Display team members in grid view.
* Select amount of columns
* Fully Responsive
* 1 layout in lite version
* Shortcodes used
* Add Social media links (Facebook, Twitter, Google+, LinkedIn and Pinterest)
* Option to add more social media links
* Super easy Installation.
* Each member has different information
* Change color styles
* Change font color
* Change font sizes
* Change outer glow on circle on hover for Each Member
* Add custom CSS
* Click on team member for full bio.
* Change number of team members.
* All major browsers supported.


Pro Version Features
= All free version features =
* Support within 12 hours
* 5 different style and layout options
* Single Site License
